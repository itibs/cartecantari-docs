Welcome to CarteCantari docs!
==================================================

Here you will find resources to start improving the `song book app`_ on your own.

You can also use the experience you gain from this app to develop other applications in Flutter.

Start with the :doc:`Zero to Hero </zero2hero>` in order to install everything you need and be able to run the application on your computer.

.. _`song book app`: https://play.google.com/store/apps/details?hl=en_US&id=ro.ber.cartecantari
